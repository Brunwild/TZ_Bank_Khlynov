[Начало]
  │
  ├─> Клиент открывает раздел "Кредиты"
  │     │
  │     ├─> Система показывает персональные предложения (предрассчитанные лимиты)
  │     │
  │     └─> Клиент выбирает "Новый кредит"
  │
  ├─> Этап 1: Заполнение заявки
  │     │
  │     ├─> Автозаполнение данных из профиля (70% полей)
  │     │
  │     ├─> Клиент вводит: сумма, срок, цель кредита
  │     │     ├─> Валидация в реальном времени
  │     │     └─> Подсказки при ошибках (e.g., "Макс. сумма 500 000 ₽")
  │     │
  │     └─> Запрос доп. документов (при необходимости)
  │           ├─> Загрузка фото/сканов
  │           └─> OCR-распознавание
  │
  ├─> Этап 2: Предварительное одобрение
  │     │
  │     ├─> Мгновенный запрос в скоринговую систему
  │     │     ├─> Одобрено → Push: "Предварительное решение положительное!"
  │     │     ├─> Отказ → Push: "Требуется дополнительная проверка"
  │     │     └─> Отклонено → SMS: "Отказ с кодом Z-103"
  │     │
  │     └─> Антифрод-проверка
  │           └─> Подозрение → Перевод в ручную обработку
  │
  ├─> Этап 3: Подписание документов
  │     │
  │     ├─> Формирование договора в ЭДО
  │     │
  │     ├─> Push: "Подпишите договор в приложении!"
  │     │
  │     ├─> Биометрическая подпись/СМС-подтверждение
  │     │     ├─> Успешно → Зачисление средств
  │     │     └─> Ошибка → "Повторите попытку"
  │     │
  │     └─> Таймер: 24 часа на подписание
  │           └─> Просрочка → Аннулирование заявки
  │
  └─> Этап 4: Выдача средств
        │
        ├─> Зачисление на указанный счет
        │     ├─> Push: "Кредит на 500 000 ₽ зачислен!"
        │     └─> SMS: "Зачисление +7********00"
        │
        └─> Создание графика платежей
              └─> Уведомление о первом платеже (за 3 дня)
[Конец]
