[Начало]
  │
  ├─> Клиент открывает раздел "Кредиты"
  │     ├─> Система определяет тип кредита: беззалоговый/залоговый
  │     └─> Показ условий для залоговых кредитов
  │
  ├─> Этап 1: Заполнение заявки
  │     ├─> При выборе залогового кредита:
  │     │     ├─> Раздел "Залоговое имущество"
  │     │     │     ├─> Выбор типа: недвижимость/авто/оборудование
  │     │     │     ├─> Автозаполнение из Росреестра (по кадастровому номеру)
  │     │     │     ├─> Ручной ввод характеристик
  │     │     │     └─> Загрузка фото документов (ПТС, выписка ЕГРН)
  │     │     └─> Автоматическая предварительная оценка
  │     │           └─> Push: "Предварительная оценка: 2 500 000 ₽"
  │     └─> ... (остальные поля)
  │
  ├─> Этап 2: Проверка залога
  │     ├─> Автоматические проверки:
  │     │     ├─> Обременения (через API Росреестра)
  │     │     ├─> Залоговая стоимость (сравнение с базами)
  │     │     └─> Юридическая чистота
  │     ├─> При сложных объектах → экспертная оценка
  │     │     └─> SMS: "Требуется осмотр объекта. Выберите время"
  │     └─> Расчет ЛТВ (Loan-to-Value)
  │
  ├─> Этап 3: Одобрение и документы
  │     ├─> Формирование двух договоров:
  │     │     ├─> Кредитный договор
  │     │     └─> Залоговое соглашение
  │     ├─> Push: "Подпишите кредитный договор и залоговое соглашение"
  │     └─> ... (процесс подписания)
  │
  ├─> Этап 4: Регистрация залога
  │     ├─> Автоматическая подача в Росреестр/ГИБДД
  │     │     ├─> Успешно → SMS: "Залог зарегистрирован"
  │     │     └─> Ошибка → уведомление юриста
  │     └─> Зачисление средств только после регистрации
  │
  └─> ... (остальные этапы)
